End-to-End Customer Churn Prediction System
Project Overview

This project demonstrates a complete machine learning workflow for predicting customer churn, from business understanding to deployment through a modular Streamlit application.

Unlike a notebook-only project, this repository contains both:

A complete exploratory data analysis and model development notebook.
A reusable end-to-end ML Pipeline application.

The project emphasizes reproducibility, modularity, and validation of results.

Validation

A major objective of this project was to verify that the automated ML application reproduces the manually developed notebook.

The comparison showed consistent performance across implementations.

Model	Notebook	Application
Logistic Regression	~50%	~48%
Decision Tree	~48%	~49%
Random Forest	~49%	~50%

These results demonstrate that the application's preprocessing, training, and evaluation pipeline is functioning correctly.
